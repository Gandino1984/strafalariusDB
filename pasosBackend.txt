PRIMEROS PASOS:
1. 
Instalar Express usando NPM(Node Package Manager). O crear el archivo package.json con
Npm init. Para crear el package.json sin rellenar usamos: 
    $ npm init -y

Agregar "type":"module" al package.json creado

Instalar Express: 
    $ npm instal express
Esto añade la dependencia de Express al archivo package.json y crea además el archivo package-lock.json con los 
detalles de las librerías instaladas y la carpeta Node Modules que hay que poner en el .gitignore de gitignore (node_modules)

Los integrantes del equipo que clonan el repositorio inicial hacen:  
    $ npm install 
porque la carpeta node modules no debe subirse al repositorio.

2. 
instalación de librerías de desarrollo que se usan durante este proceso y no en el despliegue de la 
aplicación que estamos desaroollando.

Nodemon: 
    $ npm install --save-dev nodemon

3. 
crear carpeta src en el repositorio y meter el index.js

4. 
añadir "main":"index.js"
añadir "start": "node scr/index.js" a los scripts del package.json
añadir "dev":"nodemon src/index.js"
    $ npm start

salida terminal:
" > nuestroproyecto@1.0.0 start
  > node src/index.js
"

************** en index.js *******************

5. 
    import express from "express";
    const app = express();

definir el puerto de la app:
    const APP_PORT = 3000; //puerto por defecto

6.
crear primera ruta usando el método get de la app:

-request tiene la información de la petición que nos hace el cliente
-response es el objeto que nos permitirá responder a la petición anterior

    modelo:
    app.get(<ruta>, (request, response)=>{ 
        res.send(<string ejemplo>);
    });

    ejemplo:
    app.get("/", (req, res)=>{ 
        res.send("Hola mundo!!");
    });


7.
crear el listener de la app en el puerto especificado

    app.listen(APP_PORT, ()=>{
        console.log("El servidor está en marcha en el puerto" + APP_PORT);
    })


8.
en la terminal:
    
    $ npm start 
    
    o

    $ npm run dev

salida terminal:
"
    >nuestroproyecto@1.0.0  
    >node src/index.start

    El servidor está en marcha en el puerto 3000 
"


9.
probar en el navegador:
    
    localhost:3000
